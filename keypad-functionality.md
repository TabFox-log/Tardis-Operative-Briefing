# Keypad Functionality

## Sections

**The Keypad** is divided into **multiple sections,** defined by their different **colors.**



**CheckLights:** \
TC = TakeOff CheckLight \
LC = Land CheckLight \
CPC = Console Pressure CheckLight \
SPC = System Pressure CheckLight \
F1C = Fail 1 CheckLight \
F2C = Fail 2 CheckLight \
F3C = Fail 3 CheckLight \
MPC = Main Panel CheckLights \
HtiC = Haltie CheckLights \
P1C = Power On CheckLight \
P0C = Power Off CheckLight \
SWC = Swooshy Switches CheckLight \
STC = Stott CheckLights \
SBC = Stabilize CheckLights \
**Toggle Resets:** \
SG2 = SafeGuard Switches \
ATl = Auto Land \
Stot = Stott Switches \
HTi = Halti Levers \
Mni = SYS/CONS Toggles \
Mp2 = Music Panel Levers \
VKn = Vibe Knob \
HTE = I-HTE-ACNE Switches \
SLL = Stabilized Lever Left \
SLR = Stabilized Lever Right \
StiL = Stygian Levers \
RroL = Retro Levers \
SwS = Swooshy Switches \
CT = Core Toggles \
MLr = Main Lever \
PWL = Power Lever \
**Power Toggles:** \
TL = Top Lights (Light Strips) \
BL = Bottom Lights \
SL = Stair Lights \
CL = Center Lights \
RL = Roof Lights \
DIM = Dims all lights \
**Console Toggles:** \
enble = Display On \
Off = Display Off \
don = Panel Display On \
doff = Panel Display Off \
Mpt = Music Panel Toggle \
DL = Disarm-Main Lever \
DEC = Dial-Empower Change (Enables Main Lever)
