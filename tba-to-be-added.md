Tooflips: Switch No.1: turns the tardis into a glorified flashlight on the outside, lighting up dim or dark areas.

All switches: Bluescreens the TARDIS.

Code pad(colorful one)

Blue Switches: TL: toggles lights above the door BL: Toggles Balcony speaker-like lights CL: Toggles Round bottom lights RL: Toggles Bright Roof lights DIM: idk SL: Toggles Stair Lights

Black Switches: SCRN: turns time column screen on/off NAV: turns nav display on/off SgB: Locks Safeguard B pedal in ready position. RSTT: Resets Timeline NPG: opens the button press History on the screen DL: Locks Throttle DEC: unlocks throttle.

Code pad (white one)

SET: activates current code PRT: prints current code on screen RST: resets code Im looking into the rest, ill put them here soon

CODES: 001: locks code pad 002: turns off time rotor lighting 003: dims time rotor 004: disables base lights 005: disables base light animation

Blue Beam Lights: lue swooshy switch, then Safeguard A

Light mode: button in the top left corner of the panel with red levers. Disable temporarily by pressing button on opposite side of panel.

Lambent mode: E, F, Stott 1. Use Retro levers to control colors. Good for Screenshots

Opening Music Panel: Black knib to the right of keypad, middle red button, then music panel handles. Close with blue button inside the panel.

Pong: Black knob, Bottom Button, Mode A.

SLIDERS:

1. Flight Stabilization: idk
2. Status Flight: idk 3: Restricted flight: enables restricted flight, DO NOT LEAVE ON AND FORGET ABOUT IT. 4: Flight: Enables/disables flight 5: Transtemporal travel: enables/disables time travel.

RESTRICTED FLIGHT.

Activate Safeguard B and Restricted flight. Press the pedal and pull the throttle after it reaches about halfway. Repeat for landing

Vortex Velocity: Knob on the lower laft corner of the Red Lever Panel. Each time pressed will increase time travel speed. Right click to decrease. Speeds: 1 1+ 1++ 1+++ MAX+\
